# spanning_scanline_zbuffer

## 使用说明
### 使用键盘控制：
  1. ‘x’:绕x轴旋转.
  2. ‘y’:绕y轴旋转.
  3. ‘+’:放大.
  4. ‘-’:缩小.
  5. 方向键‘↑’或数字‘8’:向上移动.
  6. 方向键‘↓’或数字‘2’:向下移动.
  7. 方向键‘←’或数字‘4’:向左移动.
  8. 方向键‘→’或数字‘6’:向右移动.
  9. ‘ESC’:退出.
